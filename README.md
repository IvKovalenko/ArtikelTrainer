# Artikel Drill — тренажёр артиклей (der/die/das)

**https://artikeldrill.com**

Тренажёр немецких артиклей для бесконечной дрилл-тренировки — без пауз, меню и
лишних кликов. Работает как сайт и ставится на телефон как приложение (PWA).

Проект родился из раздражения существующими тренерами артиклей: одни неудобные,
у других медленные анимации, у третьих слова разбиты по темам — приходится
прыгать между группами и тратить время. Здесь идея другая: открыл — и сидишь
отвечаешь слово за словом, а всё остальное приложение решает само.

Что делает его удобным:

- **Система весов.** Слова, на которые отвечаешь правильно, показываются всё
  реже; слова с ошибками — чаще, пока артикль не запомнится.
- **Новые слова подсовываются постепенно.** Начинаешь с A1; когда уровень
  освоен на ≥95%, открывается следующий (до C2).
- **Никаких пауз.** Ответил → мгновенно увидел результат и перевод → следующее
  слово. С клавиатуры: `1` = der, `2` = die, `3` = das, `4` = Plural,
  `Пробел`/`Enter` — дальше.
- **~4700 слов** — базовые уровни из официальных Wortliste Goethe-Institut
  (A1, A2, B1), а B2–C2 дополнены существительными из [немецкого Викисловаря](https://de.wiktionary.org)
  (по списку [german-nouns](https://github.com/gambolputty/german-nouns), лицензия CC BY-SA),
  отобранными по частотности в [Лейпцигском корпусе](https://wortschatz.uni-leipzig.de/de/download)
  (deu_news_2023). У каждого слова есть русский и английский перевод
  (показывается тот, что соответствует языку интерфейса: русский — при
  русском, английский — при английском и немецком).
- **Статистика и синхронизация.** Прогресс привязан к аккаунту и общий на всех
  устройствах; в приложении — таблица по каждому слову (верно/ошибки/показы).

Регистрация нужна только чтобы не потерять прогресс: подойдёт **любой
выдуманный email и пароль** (подтверждения по почте пока нет — главное, запомни
их).

## Как это работает

- Статистика хранится одним JSON:
  `{ "Haus": { "correct": 0, "wrong": 0, "seen": 0 }, ... }`.
- Клиент работает **офлайн-first**: пишет в `localStorage`, а в фоне
  синхронизирует с сервером. При старте данные с сервера и локальные
  объединяются поэлементным максимумом (прогресс не теряется).
- Следующий уровень открывается, когда на текущем освоено ≥95% слов (доступны
  только слова из открытых уровней).

### Выбор следующего слова

Слово выбирается **не равномерно**, а взвешенным случайным способом: у каждого слова есть
«вес» — чем он больше, тем выше шанс, что покажут именно это слово. Так тренажёр сам чаще
подсовывает то, что тебе тяжелее даётся, и почти не тратит время на давно выученное.

Вес слова зависит от статистики ответов по нему:

- **новое** слово (ещё ни разу не показывали) — вес **5**: новые слова вводятся охотно;
- **хорошо выученное** (3+ верных ответов подряд и ни одной ошибки) — вес **0.1**: повторяется
  очень редко, только чтобы не забылось;
- **все остальные** — вес считается по формуле `1 + ошибки×3 − верные×0.5` и не опускается
  ниже **0.2**. То есть каждая ошибка резко повышает частоту показа, а каждый верный ответ —
  постепенно понижает.

Как из весов получается конкретное слово (принцип «рулетки»):

1. Берётся пул слов из всех открытых уровней; из него исключается слово, показанное прямо
   перед этим, чтобы одно и то же не выпадало два раза подряд.
2. Складываются веса всех слов пула — получается общая сумма.
3. Выбирается случайная точка от 0 до этой суммы, и слова «проходятся» по очереди, вычитая
   их веса из точки; на каком слове точка ушла в ноль — то и показывается.

Благодаря этому слово с весом 5 выпадает примерно в 25 раз чаще слова с весом 0.2 — но у любого
слова шанс всё равно ненулевой, поэтому подборка не зацикливается на нескольких словах.

## Правила словаря

Принципы, по которым слова добавляются, сортируются и прореживаются
(`tools/gen_words.py` + переводы в `tools/translations.py` / `tools/translations_en.py`;
после любой правки — `python tools/gen_words.py`).

1. **Уровень — по употребительности.** Чем частотнее слово, тем ниже уровень
   (A1 — самое базовое, C2 — редкое). У нового слова обязателен русский и
   английский перевод; ключ перевода — точная строка слова.

2. **Композиты почти не нужны.** Составное существительное наследует род
   последней части (die Zeit → die Bauzeit), поэтому прозрачные композиты не
   дают новой информации для тренировки артикля. Правило: **на каждое базовое
   слово — не больше одного прозрачного композита на уровень** (пример для
   Zeit: A2 Freizeit, B1 Teilzeit, B2 Jahreszeit, C1 Arbeitszeit, C2 Bauzeit).
   Лишние прозрачные композиты не удаляются, а переносятся в блок
   `RESERVE_COMPOUNDS` в gen_words.py — build() его не читает, переводы
   сохраняются, вернуть слово = перенести строку обратно в активный список.

3. **Производные с приставками — НЕ композиты.** Слова вида Ausgang, Eindruck,
   Einfluss, Vorschlag, Rücksicht, Vorstellung, Überfall, Einfall образованы
   приставкой от глагола/корня и являются самостоятельными лексемами с
   собственным значением. Правило «1 на уровень» на них не распространяется —
   они остаются в словаре без ограничений.

4. **Лексикализованные композиты защищены.** Если значение слова не выводится
   из суммы частей, оно остаётся сверх лимита: Hochzeit (свадьба), Vorteil /
   Nachteil, Straßenbahn, Autobahn, Kofferraum, Feuerwerk, Handwerk,
   Kinderwagen, Datenbank, Wirbelsäule, названия элементов (Wasserstoff,
   Sauerstoff, Kohlenstoff) и т.п.

5. **Омонимы (Teekesselchen)** — одинаково пишутся и звучат, но артикль меняет
   значение (die Steuer «налог» / das Steuer «руль»). Каждое значение — своя
   карточка в списке `HOMONYMS` с gloss (русское значение) и en. gloss — часть
   ключа прогресса, менять его нельзя. Никогда не хранить такие слова одной
   карточкой с одним артиклем.

6. **Pluralia tantum** (Eltern, Geschwister, Ferien…) — в `PLURAL_WORDS`,
   правильный ответ «Plural». Только настоящие: если единственное число
   существует, слово идёт в обычные списки.

7. **Диалектизмы** (гельветизмы/австрицизмы: Paradeiser, Trottoir, Velo…) — в
   `DIALECT_VOCAB_ADD`, все на уровне C2, в переводе пометка региона
   («(австр.)» / «(швейц.)»). Критерий: у слова есть стандартный аналог, а сами
   немцы его часто не знают. Общеизвестные южные слова (Semmel, Knödel,
   Fasching) остаются в обычных списках.

8. **Сокращения и дублеты** — одной карточкой «Vollform (Kurzform)»: полная
   форма сначала, сокращение в скобках — Abonnement (Abo), Kriminalroman
   (Krimi). Объединять только формы одного слова с одним родом;
   лексикализованные усечения (Auto, Kino, Bus) — самостоятельные слова.
